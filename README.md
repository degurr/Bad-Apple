# 🍎 배드애플(Bad Apple!!) 자바스크립트 플레이어

비디오 프레임을 실시간으로 분석하여 입자로 재구성하는 인터랙티브 아트 프로젝트입니다.

- **제작**: simsimman
- **참고 영상**: [Bad Apple!! But it’s a JavaScript Tutorial](https://www.youtube.com/watch?v=juIFchliumI) (by Radu Mariescu-Istodor)

## 🛠 주요 조작법
1. **재생바**: 하단 바를 마우스로 조절하여 원하는 장면으로 이동할 수 있습니다.
2. **문자 설정**: 화면을 채울 한 글자를 입력하세요. (한글, 영어, 숫자, 이모지 모두 가능)
3. **크기(RES)**: 입자의 크기를 결정합니다. 숫자가 작을수록 고해상도 아트로 변합니다.
4. **사진 업로드**: `사진 업로드` 버튼으로 본인의 이미지를 활용한 픽셀 아트를 만들 수 있습니다.

## ⌨️ 단축키
- **Space**: 재생 / 일시정지
- **M**: 음소거 토글
- **R**: 처음부터 다시 시작
